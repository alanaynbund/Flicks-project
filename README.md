# Flicks
Share your flicks! Upload pictures, like others, delete them, and edit them! Share your favorite flicks to everybody!

![](./video/demo.gif)

# Technology Used

* React
* Mongoose
* JS
* Dotenv
* Bcrypt
* OAuth
* Express
* Node
* Jsonwebtoken
* Cors
* Moment
* Axios
* MaterialUI

Setup:
- run ```npm i && npm start``` for both client and server side to start the app
